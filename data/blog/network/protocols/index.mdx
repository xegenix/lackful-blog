Index of Network Protocols

# IP - Internet Protocol
- IPv4 - Internet Protocol Verson 4 is a legacy network address system consisting of a 32bit address used to route information across the public internet, as of 2019 all available version 4 addresses have been depleted. 
- IPv6 - Internet Protocol Version 6 is a 128bit addressing system designed to replace version 4 as a result of the depleting IPv4 address space.
